## Getting Started

This is Folder Management application

### Prerequisites

- Node.js (v16.19.1) and npm
- Git

### Installing

Clone the repository and install dependencies.

```bash
git clone git@github.com:cuong27102017/fe-folder.git
cd fe-folder
npm install
```

### Testing

Run all tests

```bash
npm test
```

### Running

Start project

```bash
npm start
```
